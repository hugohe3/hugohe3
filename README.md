## 🚀 Featured Project

### [`ppt-master`](https://github.com/hugohe3/ppt-master) — AI generates natively editable PPTX from any document

[![Stars](https://img.shields.io/github/stars/hugohe3/ppt-master?style=social)](https://github.com/hugohe3/ppt-master/stargazers)
[![Forks](https://img.shields.io/github/forks/hugohe3/ppt-master?style=social)](https://github.com/hugohe3/ppt-master/network/members)
[![License](https://img.shields.io/github/license/hugohe3/ppt-master?style=flat-square)](https://github.com/hugohe3/ppt-master/blob/main/LICENSE)

Drop in a **PDF, DOCX, URL, or Markdown** — get back a **natively editable PowerPoint** with real shapes, text boxes, and charts. Not images. Click anything and edit it.

- 🎨 **Real PowerPoint** — DrawingML shapes, directly clickable and editable
- 💰 **Transparent cost** — as low as **$0.08/deck** with VS Code Copilot
- 🔒 **Data stays local** — only the AI model call leaves your machine
- 🔌 **No lock-in** — works with Claude Code, Cursor, VS Code Copilot, Codebuddy; supports Claude, GPT, Gemini, Kimi

**[Live Demo](https://hugohe3.github.io/ppt-master/)** · **[Examples (15 projects · 229 pages)](https://github.com/hugohe3/ppt-master/tree/main/examples)** · **[Why PPT Master?](https://github.com/hugohe3/ppt-master/blob/main/docs/why-ppt-master.md)**
